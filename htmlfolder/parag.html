The github.dev editor introduces a lightweight editing experience that runs entirely in your browser. With the github.dev editor, you can navigate files and source code repositories from GitHub, and make and commit code changes. You can open any repository, fork, or pull request in the editor. The github.dev editor is available to everyone for free on GitHub.com. The github.dev editor introduces a lightweight editing experience that runs entirely in your browser.
